# Example typescript CrossCode mod (compliation using esbuild)

## Building

```bash
git clone https://github.com/krypciak/cc-ts-template-esbuild
cd cc-ts-template-esbuild
npm install
npm run start
# this should return no errors (hopefully)
npx tsc
```
