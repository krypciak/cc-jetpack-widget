# Jetpack widget

Adds a jetpack widget to [CCUILib](https://github.com/conorlawton/nax-ccuilib)  

## Building

```bash
git clone https://github.com/krypciak/cc-jetpack-widget
cd cc-jetpack-widget
npm install
npm run start
# this should return no errors (hopefully)
npx tsc
```
